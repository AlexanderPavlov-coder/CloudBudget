Differance between Bootstrap 3 and 4?

Currently, Bootstrap 4 is the latest version of Bootstrap.

In BS3 its

 - using flot system for grid system.
 - Css written in Less
 - 4 tier grid system (xs, sm, md, lg)
 - 14 px global font
 - unit pixels

In BS4 its

 - using Flex Modal for grid system.
 - Css written in Sass
 - 5 tier grid system (xs, sm, md, lg, xlg)
 - 16 px global font
 - unit rem (for mobile using px)
 - you can create a card
and much more!!!
